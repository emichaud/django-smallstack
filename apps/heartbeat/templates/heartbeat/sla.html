Monitoring Interval

The heartbeat system checks service availability every {{ expected_interval }} second{{ expected_interval|pluralize }}. Each check records an "ok" or "fail" status along with the response time.

Uptime Calculation

uptime % = (ok checks / expected checks) × 100

  • Expected checks = elapsed seconds since monitoring start / {{ expected_interval }}s interval
  • OK checks = heartbeats with "ok" status in the window
  • Missed checks (no heartbeat recorded) count against uptime

SLA Thresholds

  • Goal ({{ sla_target }}%) — internal target. Allows up to {{ target_down_monthly }} min downtime/month.
  • Commitment ({{ sla_minimum }}%) — breach threshold. Allows up to {{ minimum_down_monthly }} min downtime/month.

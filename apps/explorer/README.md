# explorer — generic CRUD browser

A registry-driven browser that exposes any registered model as CRUD pages at
`/smallstack/explorer/` without writing a per-model view.

**Status:** Framework-provided — don't edit in downstream forks; register your models with it
instead.

**Key files:** `registry.py`, `views.py`. **URL:** `/smallstack/explorer/`.

**See:** [`../../docs/skills/explorer.md`](../../docs/skills/explorer.md).
